
### 前端开发工具总结
工欲善其事必先利其器，mac对于前端开发者而言，个人感觉是非常好用的，出色的屏幕，易于配置的环境，好用的命令行工具，对于软件而言，同样也需要仔细甄选，下面总结些比较好用软件。


### 实用软件

1. VS code 超级流行好用的编辑器，微软开发，开源免费，插件多
2. iterm2 免费的终端神器, 搭配zsh和oh-my-zsh，好看又好用
3. postman 模拟请求，查看返回，接口调试必备
4. SwitchHosts! 一键切换hosts，测试环境，开发环境，切换
5. charls 抓包神器，抓包改包，改转发，查看请求
6. spy-debugger 超级推荐神器，webview页面调试利器
7. ps插件-cutterman 一键切图简单高效
8. ps插件-pxCook  psd导出标注，自动切图
9. sketch 偏向设计的前端必备设计软件，ui还原，真机查看
 

### chrome 插件
1. gitlabtree diff代码的时候超级实用，目录，改动文件清晰明了
2. QR Code Generator 二维码生成，方便调试
3. google翻译  快速翻译，代码，学习两不误
4. Vue devtools vue开发调试工具
5. React Developer Tools react开发调试工具




以上为个人开发总结，欢迎大家修正 ，补充。
